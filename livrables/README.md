# Livrables

Ce dossier contient tout ce que Claude produit pour Ryan.

## Règle d'or

- **Inputs** (documents que tu fournis à Claude) → `context/import/`
- **Outputs** (ce que Claude produit pour toi) → `livrables/`

## Organisation

| Dossier | Contenu |
|---------|---------|
| `sites-web/` | Sites internet : landing pages, portfolios, wireframes, copies |
| `applications/` | Outils, scripts, automatisations, bots |
| `youtube/` | Briefs vidéos, scripts, hooks, calendriers éditoriaux |

## Convention de nommage

Format : `AAAA-MM-JJ_nom-du-projet_version`

Exemples :
- `2026-06-21_landing-page-jeu-de-cartes_v1`
- `2026-06-21_script-youtube-trading_v2`
- `2026-06-21_bot-alerte-memecoin_v1`

La date en préfixe permet de trier chronologiquement et de retrouver facilement ce qui a été produit lors d'une session donnée.
